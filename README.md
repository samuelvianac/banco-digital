# Banco Digital em Java

Projeto desenvolvido em Java para praticar Programação Orientada a Objetos no contexto bancário.

---

##  Objetivo

Reproduzir uma solução de banco digital aplicando os principais conceitos de POO:

- Abstração  
- Encapsulamento  
- Herança  
- Polimorfismo  

---

##  Estrutura do Projeto

```text
src/
 ├── Banco.java
 ├── Cliente.java
 ├── Conta.java
 ├── ContaCorrente.java
 ├── ContaPoupanca.java
 └── Main.java
```

---

##  Funcionalidades

- Criação de cliente  
- Criação de conta corrente  
- Criação de conta poupança  
- Depósito  
- Saque  
- Transferência entre contas  
- Impressão de extrato  

---

##  Tecnologias utilizadas

- Java  
- Programação Orientada a Objetos  
- GitHub  

---

##  Execução

Ao rodar o programa, será possível visualizar:

- Criação de contas  
- Movimentações financeiras  
- Impressão de extratos
