# Banco Digital em Java

Projeto desenvolvido em Java para praticar Programação Orientada a Objetos no contexto bancário.

## Objetivo

Reproduzir uma solução de banco digital com os principais conceitos de POO, como:

- abstração
- encapsulamento
- herança
- polimorfismo
